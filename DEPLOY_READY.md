## 🚀 Deploy to Vercel (Manual Steps - 2 minutes)

### Option 1: Via Vercel Dashboard (Recommended)

1. **Go to**: https://vercel.com/new/git/import
2. **Paste Repository URL**: `https://github.com/elrepositoriodeloabsurdo/lambert-coffee-store`
3. **Click** "Continue"
4. **Vercel will auto-detect**:
   - Framework: Next.js / Vite ✓
   - Build Command: `npm run build` ✓
   - Output Directory: `dist` ✓
5. **Set Environment Variables** (optional):
   - `VITE_WHATSAPP_NUMBER` = your phone number
   - `VITE_TUU_CHECKOUT_URL` = your checkout link
   - `VITE_TRANSBANK_CHECKOUT_URL` = your link
6. **Click "Deploy"**
7. **Wait 3-5 minutes** - Your site will be live!

### Option 2: Automatic GitHub Integration

1. Go to https://github.com/settings/applications/new
2. Create OAuth App:
   - App name: `Lambert Coffee Store`
   - Homepage URL: `https://vercel.com`
   - Authorization callback URL: `https://vercel.com/callback`
3. Connect to Vercel
4. Done! Future pushes auto-deploy
